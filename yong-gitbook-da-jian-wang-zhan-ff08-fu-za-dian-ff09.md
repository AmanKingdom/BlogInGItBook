一、下载安装 [node.js](https://nodejs.org/en/)

![](/assets/import31.png)

二、安装 gitbook 命令行工具 gitbook-cli

```
npm install gitbook-cli -g
```

> 现在下载的 node.js 自带了 npm 工具包了，所以不需要另外下载 npm



三、创建 gitbook

在 cmd 下，我们创建一个目录来存放 gitbook 即可：
